<objective>
Install the `server-only` npm package and add `import 'server-only'` as the first line of `lib/supabase-admin.ts` and `lib/email.ts`. This causes Next.js to throw a build-time error if either file is imported in a client component, preventing the service role key and Resend API key from leaking into client bundles.

Purpose: The service role key bypasses Supabase Row Level Security entirely — if it reaches the client bundle, any user can extract it and read or write any data in the database. Build-time enforcement via `server-only` is the standard Next.js App Router pattern and catches accidental client imports before they ship.

Output: Two modified lib files with server-only guards; `server-only` in package.json dependencies.
</objective>
