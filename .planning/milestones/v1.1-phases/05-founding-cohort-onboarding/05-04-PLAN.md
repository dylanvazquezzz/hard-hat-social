    **1. Overview** — One paragraph: what this doc is, who it's for, what you'll accomplish by the end.

    **2. Pre-Flight Checklist (Do These in Order)**

    Must cover these steps in sequence — order matters because env var changes require a Vercel redeploy before they take effect:

    Step 1: Set `NEXT_PUBLIC_APP_URL`
    - Where: Vercel dashboard → your project → Settings → Environment Variables
    - Value: `https://contractors-connect.vercel.app` (the actual production URL — no trailing slash)
    - Why first: Every email sent (approval, rejection) reads this var. All links in emails point here. Setting it wrong means welders get emails with localhost links. Trigger a Vercel redeploy after setting.

    Step 2: Set `NEXT_PUBLIC_CONTACT_EMAIL`
    - Where: Vercel dashboard → same Settings → Environment Variables
    - Value: your personal email (e.g., `dylan@example.com`)
    - Why: Shows in the invite-only banner on /apply so welders can reach you. Can change this anytime — just needs a redeploy.
    - Trigger a Vercel redeploy after setting (can batch with Step 1's redeploy if done at same time).

    Step 3: End-to-end test (before any real outreach)
    Create a dummy test account and run the full flow:
    1. Create a new account at `https://contractors-connect.vercel.app/auth` with a throwaway email (e.g., a Gmail +alias or a temp email service)
    2. Submit an application at `/apply` — use fake data, no real documents needed
    3. Go to `/admin` and approve the test application
    4. Check your throwaway email inbox — the approval email should arrive with a "Sign In Now" button. Inspect the href: it must point to `https://contractors-connect.vercel.app/auth?redirect=/profile` (not localhost)
    5. Click the button — sign in — confirm you land on `/profile`
    6. Confirm the welcome banner appears ("Welcome to Contractors Connect — complete your profile to appear in the directory")
    7. Upload a profile photo — confirm the welcome banner disappears
    8. Search the directory at `/contractors` — your test contractor profile should appear
    9. Only proceed to outreach once all 8 steps pass

    **3. Outreach Message Templates**

    Note: welders need time to gather documents before applying — AWS cert, insurance proof, state license. Outreach message should mention this so they're prepared before clicking /apply.

    Template A — Direct Text/DM (personal contacts):
    ```
    Hey [Name], building something I think you'd want to be part of.

    It's a verified welding contractor directory — kind of like LinkedIn but only for tradespeople who can prove their creds. You'd be one of the first on it.

    Apply here: https://contractors-connect.vercel.app/apply

    You'll need: AWS cert or equivalent, state license (if you have one), and proof of general liability insurance. Scan or photo is fine — takes about 5 minutes.

    Let me know if you have questions.
    ```

    Template B — Group Chat/Trade Group Post:
    ```
    Launching Contractors Connect — a verified-only directory for credentialed welders and tradespeople. No unverified accounts, no spam.

    Applying is invite-only right now. If you're a certified welder with proof of insurance:
    https://contractors-connect.vercel.app/apply

    You'll need: AWS cert / equivalent, state license, insurance docs ready to upload.
    ```

    **4. After Each Application Arrives**

    When you get a new application in the admin queue (`/admin`):
    - Review the submitted documents (they appear as links in the application row)
    - Approve if credentials are valid — approval email is sent automatically
    - Reject if insufficient docs — rejection email is sent with a reapply link
    - After approval, visit `/admin/contractors/[id]` to review the auto-created certification record. Edit it if the name or issuing body needs correction (e.g., change "Welding Credential" to "AWS Certified Welder D1.1").

    **5. Notes**

    - The founding cohort will likely take 1-3 weeks to get their documents together — set realistic expectations in your outreach
    - Direct the first 20-50 to the platform. Once they're on it, the directory will have enough density to be useful
    - Contact info (phone/email) is only visible to approved contractors who are logged in — welders can share safely
    - If anyone asks "is this free?" — yes, completely free for the founding cohort
