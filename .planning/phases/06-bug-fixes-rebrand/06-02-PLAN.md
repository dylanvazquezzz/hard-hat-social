
    CRITICAL — Tailwind class name rule: Every class using these tokens must be written as a complete static string in JSX (e.g., `bg-brand-blue`, `text-brand-yellow`). Never construct class names dynamically with template literals (e.g., `bg-brand-${color}`) — Tailwind's content scanner cannot detect them and they will be purged from the CSS output.

    The content array stays exactly as-is. Do not change plugins.
